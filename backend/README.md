# Tournament Backend API

This is the backend API for the Gaming Tournaments application. It handles player registrations, tournament management, and score tracking with a robust, modular setup.

## Built With
- Node.js
- Express
- MongoDB (Mongoose)
- Joi (Input Validation)

## Getting Started

1. Set up your `.env` based on `.env.example`.
2. Run `npm install` to install dependencies.
3. Start the server using `npm start` or `npm run dev`.

## Features
- Fully documented modular architecture
- Complete error handling with custom `ApiError`
- Request validation with Joi
